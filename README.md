# 📊 California Sales Management System | Power BI Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze California sales data. It provides business insights through KPIs, charts, and dynamic filters to support data-driven decision-making.

## 🚀 Features
- Sales Performance Dashboard
- Revenue & Profit Analysis
- Product-wise Sales Analysis
- Category-wise Sales Analysis
- Customer Segment Analysis
- Region-wise Sales Insights
- Year & Month Slicers
- Interactive KPIs
- Dynamic Visualizations

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Microsoft Excel

## 📈 KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Sales
- Top Products
- Sales by Category
- Sales Trend

## 📂 Files Included
- California Sales Management System.pbix
- Dataset (.xlsx/.csv)
- Dashboard Screenshots

## 🎯 Objective
To transform raw sales data into an interactive business intelligence dashboard that helps analyze sales performance, identify trends, and support better business decisions.
